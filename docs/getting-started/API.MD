## Utilisation

For config ngx-magic table is simple and It's magic!

Actually, there are three inputs to component:

* Columns,
* Data,
* tableOptions;

```xml
<ngx-magic-table [columns]="columns" [data]="data" [tableOptions]="tableOptions"> </ngx-magic-table>
```
